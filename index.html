class wutil.WMArray¶

Dynamically resized array. The interface is consistent with Python lists.

append(self, item)¶

Add item to the end of the array. Wrapper around WMAddToArray.

clear(self)¶

Remove all elements of the array. Wrapper around WMEmptryArray.

copy(self)¶

Returns a copy of the array. Wrapper around WMCreateArrayWithArray.

count(self, item)¶

Return the number of instances of item in the array. Wrapper around WMCountInArray.

extend(self, WMArray other)¶

Add all elements of other at the end of the array. Wrapper around WMAppendArray.

insert(self, int index, item)¶

Add item to the array at position index. Wrapper around WMInsertInArray.

pop(self)¶

Remove and return the item at the end of the array. Wrapper around WMPopFromArray.

class wutil.WMArrayIterator(WMArray array)¶

Iterator for WMArray objects. The next method is a wrapper around WMArrayFirst and WMArrayNext.
